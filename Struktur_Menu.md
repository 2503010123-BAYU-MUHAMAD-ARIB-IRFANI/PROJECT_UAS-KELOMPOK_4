# Struktur Menu - Sistem Manajemen Transportasi MRT/KRL

## MENU UTAMA
```
=== SISTEM MANAJEMEN MRT/KRL ===
1. Manajemen Stasiun
2. Manajemen Jadwal Perjalanan
3. Perpindahan Jalur
4. Pencarian Rute
5. Data Penumpang
0. Keluar
```

---

## 1. Manajemen Stasiun (Ade)
```
=== MANAJEMEN STASIUN ===
1. Tambah Stasiun
2. Tampilkan Semua Stasiun
3. Cari Stasiun
4. Hapus Stasiun
0. Kembali ke Menu Utama
```
